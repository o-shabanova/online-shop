# Project template

## Quick Start

**Prerequisites:** Node.js 18+ (з npm)

```bash
# Встановити залежності
npm install

# У терміналі №1: компіляція Sass у watch-режимі
npm run watch:css

# У терміналі №2: запустити локальний сервер (обслуговує папку src/ та відкриває проєкт)
npm run start


# Відкрий у браузері
http://localhost:5173
